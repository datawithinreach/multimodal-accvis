# Experiments for Visualization Accessibility Strategies


## Run & Deploy

**Installation**
```bash
npm install
```

**Run on a local development server**
```bash
npm start
```


**Deploy on Github Pages** using [gh-pages](https://www.npmjs.com/package/gh-pages)
```bash
npm run build
npm run deploy
```
